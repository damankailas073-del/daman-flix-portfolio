# Daman Kailas - Production Assistant Portfolio

A modern, Netflix-inspired portfolio website showcasing production assistant experience and skills in Film & TV.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- GSAP (animations)

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```sh
npm run build
```

## Project Structure

- `/src/components` - Reusable UI components
- `/src/pages` - Main page components
- `/src/assets` - Images and static assets
- `/src/lib` - Utility functions

## Features

- Responsive Netflix-inspired design
- Animated text with GSAP
- Interactive skill cards
- Experience timeline
- Contact section with social links
